# Guardian 100 Novels Tracker

A static checklist for the Guardian's 100 best novels list.

It stores each person's selections in their own browser with `localStorage`.
There is no server, database, signup, or analytics.

## Publish on GitHub Pages

1. Create a new GitHub repository.
2. Upload `index.html`, `styles.css`, `app.js`, `README.md`, and `.nojekyll`.
3. In the repository, open `Settings` -> `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select the `main` branch and `/root`, then save.

GitHub will publish the site at a URL like:

```text
https://YOUR-USERNAME.github.io/YOUR-REPOSITORY/
```

## Local preview

Open `index.html` in a browser. No build step is needed.
